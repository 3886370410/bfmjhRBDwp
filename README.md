# Python豆瓣电影数据分析与可视化系统

## 核心功能
- 数据采集：通过网络爬虫技术抓取豆瓣电影数据
- 数据存储：使用MySQL数据库存储电影数据
- 数据分析：对电影数据进行统计分析
- 数据可视化：利用Echarts可视化库进行数据展示

## 适用场景
- 电影爱好者：了解电影评分、评分趋势等数据
- 电影行业从业者：分析电影市场趋势，为决策提供依据
- 教育科研：用于数据分析和可视化教学

## 技术特点
- 使用Python语言进行开发，代码可运行
- 基于Flask框架搭建Web应用
- 集成Echarts可视化库，实现数据可视化
- 利用requests爬虫框架抓取豆瓣电影数据
